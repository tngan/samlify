# Advanced

This chapter covers advanced scenarios. Real-world deployments often require customization; the library aims to remain simple while exposing enough configuration hooks to support those scenarios. The chapters are organized step by step, mirroring the structure of the basic tutorial.

- Custom templates for SAML requests and responses
- Scaling with multiple identity and service providers
- Additional assertion validations
- Metadata distribution
- Configuring identity and service providers without metadata

If you have a use case that is not covered here, please open an issue on the [GitHub repository](https://github.com/tngan/samlify).
